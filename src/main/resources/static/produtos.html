Cadastrar Item
        Passo 1: Abra a página de cadastro

        Clique na opção "Cadastro" no menu principal.
        Passo 2: Preencha as informações do item

        Insira o nome do item no campo "Nome".
        Selecione a categoria do item no menu suspenso "Categoria".
        Caso a categoria desejada não esteja disponível, clique em "Adicionar Categoria" para abrir um modal onde você
        pode inserir uma nova categoria.
        Selecione a marca do item no menu suspenso "Marca".
        Se a marca desejada não estiver listada, clique em "Adicionar Marca" para abrir um modal onde você pode inserir
        uma nova marca.
        Insira o preço do item no campo "Preço".
        Insira a quantidade disponível em estoque no campo "Quantidade em estoque".
        Passo 3: Salve o item cadastrado

        Clique no botão "Cadastrar Produto" para salvar as informações do item no banco de dados.
        Após seguir esses passos, o item será cadastrado no sistema. Certifique-se de preencher todas as informações
        necessárias corretamente antes de salvar.
Consultar Item
        Acesse a página de consulta
		   

        No menu principal, clique na opção "Estoque" para abrir a página de estoque.
        Visualize a lista de produtos

        Na página de estoque, será exibida uma tabela com os produtos cadastrados.
        A tabela contém as seguintes colunas: ID, Produto, Categoria, Marca, Preço e Quantidade em Estoque.
        Localize o item desejado na lista

        Role a tabela para cima ou para baixo para encontrar o item que você deseja consultar.
        Caso haja muitos itens na lista, você pode utilizar a funcionalidade de pesquisa ou ordenação para facilitar a
        busca.
        Analise os detalhes do item

        Na linha correspondente ao item desejado, verifique as informações presentes em cada coluna.
        Observe o nome do produto, a categoria à qual pertence, a marca, o preço e a quantidade em estoque.
        Volte à página de estoque

        Para realizar consultas adicionais ou visualizar outros produtos, você pode utilizar o botão "Voltar" ou navegar
        novamente até a página de estoque.
        Ao seguir esses passos, você poderá consultar os itens disponíveis no estoque. Lembre-se de localizar o item
        desejado na lista e analisar os detalhes relevantes antes de prosseguir com outras ações.
Como faço para atualizar as informações de um produto cadastrado?
        Para atualizar as informações de um produto, acesse a página de estoque e localize o item desejado na lista.
        Clique no botão de edição correspondente ao produto e faça as alterações necessárias nos campos disponíveis. 
        Emseguida, salve as modificações para atualizar os dados no banco de dados.
É possível excluir um produto do estoque?
          Sim, é possível excluir um produto do estoque. 
          Na página de estoque, localize o produto que você deseja excluir na tabela e clique no botão de exclusão correspondente. 
          Será exibida uma mensagem de confirmação e, ao confirmar a exclusão, o produto será removido permanentemente do banco de dados.
Como posso adicionar novas categorias ou marcas?
          Para adicionar uma nova categoria, clique no botão "Adicionar Categoria" ao selecionar a categoria de um produto no formulário de cadastro. Isso abrirá um modal onde você poderá inserir o nome da nova categoria.
          Da mesma forma, para adicionar uma nova marca, clique no botão "Adicionar Marca" ao selecionar a marca de um produto no formulário de cadastro. Um modal será aberto, permitindo que você insira o nome da nova marca.
